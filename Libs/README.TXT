SharpDL is a game-focused library that offers XNA-like game loops and 2D rendering on top of SDL2.

Requirements:

- Windows
	- SDL2#.dll compiled against .NET Framework 4.5+
	- libfreetype-6.dll
	- libpng16-16.dll
	- SDL2.dll
	- SDL2_image.dll
	- SDL2_ttf.dll
	- zlib1.dll
- Linux
	- SDL2#.dll compiled against Mono / .NET 4.5+
	- libfreetype.so.6 (compiled for specific distros)
	- libpng12.so.0 (compiled for specific distros)
	- libSDL2-2.0.so.0 (compiled for specific distros)
	- libSDL2_image-2.0.so.0 (compiled for specific distros)
	- libSDL2_ttf-2.0.so.0 (compiled for specific distros)
	- libz.so.1 (compiled for specific distros)
